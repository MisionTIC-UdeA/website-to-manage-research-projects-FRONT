# Website to manage research projects - FRONTEND

## Modules

The project has 4 modules:
* Users
* Projects
* Inscriptions 
* Advances

## How to use this project?

**Step 1: Clone the project.**

### `git clone`

**Step 2: Navigate to the folder and install dependencies.**

### `yarn install`

**Step 3: Run the project.**

### `yarn start`
